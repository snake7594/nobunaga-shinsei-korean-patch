# 번역 데이터

노부나가의 야망 신생(信長の野望・新生, 1.1.4) 한글 번역 데이터입니다.

- `source_jp/`  — 게임에서 추출한 원문 문자열 배치 (번역 키). 529개 파일.
- `korean/`     — index로 짝지어진 한국어 번역. 529개 파일.

## 파일 형식

원문 `source_jp/bXXXX_*.json`:
```json
{ "cat": "dialog",
  "items": [ { "i": 0, "t": "원문 문자열" }, ... ] }
```

번역 `korean/bXXXX_*.json`:
```json
{ "items": [ { "i": 0, "t": "번역 문자열" }, ... ] }
```

- `i` 인덱스로 원문↔번역이 대응됩니다.
- 서식 토큰은 보존해야 합니다: `%s %d`, `\n`(줄바꿈), `[bNNN]`류 이름 플레이스홀더,
  `<ESC>C?`(U+001B) 컬러 태그.
- 적용은 `tools/apply_translations.py` 가 원문 문자열을 키로 매칭해 MSG 파일에 주입합니다.

## 품질

- 기계 번역이 상당수 포함되어 있으며 일부 오역·어색함이 있을 수 있습니다.
- 인명·지명은 국립국어원 일본어 표기법을 지향합니다.
- 개선/기여 환영합니다. `korean/` 의 `t` 값을 고쳐 PR 하시면 됩니다.

## 고지

- 팬 번역이며 코에이 테크모와 무관합니다.
- 원문(`source_jp/`)은 번역 작업용 참조 키로, 게임 원본 데이터·에셋은 포함하지 않습니다.
