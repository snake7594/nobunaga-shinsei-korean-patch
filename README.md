# 노부나가의 야망 신생 한글 패치 (비공식) — v2.4

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.

게임 버전은 **일본 일반판 1.1.4**입니다. DLC는 아직 적용하지 않았습니다.

폰트에 한글 2,400여 자를 추가하고 게임 텍스트를 한국어로 번역합니다.

> **v2.4 — 일부 창에서 글자가 가로로 늘어져 보이던 문제 수정**
> 게임에는 글자를 그리는 방식이 두 가지 있는데(가변폭/고정칸), v2.3의 자간 최적화가
> 고정칸 방식 창에서 글자를 가로로 늘어뜨리는 부작용이 있었습니다. v2.4는 글리프
> 원본은 그대로 두고 간격 정보만 조정하는 방식으로 바꿔 **두 방식 모두 정상**입니다.
> (대사창 자간 최적화 효과는 유지)
>
> **v2.3 — 시나리오 대사 줄바꿈 문제 해결 + 한글 자간 최적화**
> - 한글 번역이 일본어보다 길어 대사가 3줄을 넘겨 잘리던 문제를 해결했습니다.
>   ① 한글 글자 간격을 실제 글자 폭에 맞게 최적화(약 20% 조밀, 훨씬 자연스러운 조판)
>   ② 전각 문자(（１５６０）？！ 등)를 반각으로 정규화해 폭 절약
>   ③ 대사 556건의 어색한 강제 줄바꿈을 제거해 게임이 최적으로 줄을 나누도록 개선
>   → 전체 대사 13,000여 건이 모두 3~4줄 안에 표시됩니다.
> - 이름 사이 가운뎃점(・)과 물음표가 ?로 깨져 보이던 문자 366건도 수정했습니다.
>
> **v2.2 — 시스템/명령 버튼과 여러 화면을 대거 한글화**
> - 시스템·내비게이션 버튼 **약 21종**(뒤로·다음·닫기·결정·예·아니오·중지·거부·부인·승인·승낙·전부개방·지휘·재교섭·처단·건너뛰기·개시·등용·무장·희·개전) — 색상 상태 포함 114개 버튼
> - 전투 **군평정** 화면(군평정·대·승리·패배·전공)
> - **튜토리얼** 흐름 다이어그램(지행·내정·전투·천하통일·세력·성·군·무장)
> - **합전 시작 배너** 9종(오케하자마·나가시노·미카타가하라·가와나카지마·이쓰쿠시마 전투 등)
> - v2.1의 **원형 명령 메뉴** 글자 크기를 원본에 맞게 조정
> 픽토그램·버튼 배경·타이틀 로고(信長の野望 상표)는 원본 그대로 보존합니다.

## 포함 내용
- 게임 텍스트 100% 번역 (메뉴·UI·이벤트 대사·게임 진행·무장 열전·튜토리얼)
- 한글 폰트 2,400여 자 (서울한강체)
- **이미지(텍스처) 한글화**: 원형 명령 메뉴 · 시스템/내비 버튼 · 전투 군평정 · 튜토리얼 다이어그램 · 합전 배너 · 화면 제목 108종 · 부팅 경고문
- 실행파일 시스템 메시지 6종 한글화 (1.1.4 기준)

## 적용 대상
- 타이틀: 노부나가의 야망 신생 (일본판), **버전 1.1.4**
- 타이틀 ID: `01007AB012872000`

## 설치 방법 (클린 설치 권장)
1. 기존 모드 폴더를 **완전히 삭제**
2. 게임 목록에서 해당 게임을 **우클릭 → Open Mods Directory**
3. zip 속 `NobunagaShinsei_KR` 폴더를 통째로 복사
4. 게임 실행

Yuzu 계열은 우클릭 → Open Mod Data Location, 실기(Atmosphère)는
`atmosphere/contents/01007AB012872000/`에 `romfs`·`exefs` 복사.

## exefs 패치 주의
- `exefs/main`(실행파일)은 **게임 버전 1.1.4 전용**입니다.
- 부팅에 문제가 생기면 `exefs` 폴더만 삭제하세요. 나머지 한글패치는 그대로 동작합니다.

## 알려진 사항
- 폰트는 서울한강체(서울시 배포 무료 글꼴)라 원본 붓글씨체와 스타일이 다를 수 있습니다.

- 기계 번역이 포함되어 일부 문장은 어색하거나 오역이 있을 수 있습니다.

## 저장소 구성 (개발/기여용)

이 저장소에는 **패치를 만드는 도구와 번역 데이터**가 함께 들어 있습니다.
게임 원본 데이터·복호화 키는 포함하지 않습니다.

```
tools/            빌드 파이프라인 (Python) — 포맷 디코드/인코드, 폰트 확장,
                  MSG 재조립, 이미지 로컬라이즈, 패키징
  extractor/      병합 romfs 추출기 (LibHac, .NET 6) — 키는 본인 것 사용
  scratch/        1회성 디버그/수정 스크립트 (참고용)
translation/
  source_jp/      추출한 원문 문자열 배치 (번역 키) 529개
  korean/         index로 짝지어진 한국어 번역 529개
docs/
  FORMATS.md      리버스 엔지니어링한 파일 포맷 분석 (LINKDATA/KT/NSO/MSG/G1N/G1T)
  BUILD.md        빌드 가이드
```

빌드 방법은 [docs/BUILD.md](docs/BUILD.md), 포맷 분석은 [docs/FORMATS.md](docs/FORMATS.md) 참고.

### 기여
번역 개선은 `translation/korean/` 의 `t` 값을 고쳐 PR 해주세요.
(서식 토큰 `%s %d \n [bNNN]`, `<ESC>C?` 컬러 태그는 보존 필수)

## 라이선스 / 고지
- **팬 번역이며 코에이 테크모와 무관합니다. 게임 원본 데이터·에셋은 포함되어 있지 않습니다.**
