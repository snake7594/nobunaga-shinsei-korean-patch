# 노부나가의 야망 신생 한글 패치 (비공식) — v3.0

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.

> **이 패치는 게임 버전 1.1.5(파워업키트 적용) 전용입니다.**
> 파워업키트가 적용되지 않은 구버전(1.1.4)을 지원하던 옛 릴리스는 저장소 히스토리에는
> 남아 있지만 더 이상 유지하지 않습니다. 앞으로는 1.1.5(파워업키트) 기준으로만 관리합니다.

폰트에 한글 2,400여 자를 추가하고 게임 텍스트(메뉴·UI·이벤트 대사·파워업키트 신규
콘텐츠 포함) 전체를 한국어로 번역합니다.

## v3.0 — 파워업키트 대사·메뉴 텍스트 완전 한글화 ★

이전 버전까지는 이미지(원형 명령 메뉴, 시스템 버튼 등)만 한글이고 **파워업키트의
대사·메뉴 텍스트는 일본어**로 나오는 문제가 있었습니다.

**원인**: 파워업키트는 일반 게임과 **별도의 실행 프로그램**(타이틀 ID
`01007AB012872001`, 일반 게임의 `01007AB012872000`과는 다름)이며, 텍스트를 일반판과
다른 전용 파일(`MSG_PK/`, `RES_JP_PK/`)에서 읽습니다. 게다가 이 PUK 프로그램은
독자적인 폰트 리소스(`res_lang_pk.bin`)를 쓰는데, 폰트 로더가 **원본과 정확히 같은
압축 해제 크기**를 요구해서 — 일반판에 쓰던 방식(폰트를 통째로 더 큰 한글 폰트로
교체)을 그대로 적용하면 **부팅 전에 죽습니다.** v3.0은 이 두 파일 세트를 모두
찾아내 정확한 방식(미사용 전각 글자 자리에 한글을 크기 불변으로 제자리 삽입)으로
한글화했습니다. 자세한 내용은 [docs/FORMATS.md](docs/FORMATS.md#7-파워업키트-872001-이중-프로그램-구조)를
참고하세요.

- 파워업키트 신규 콘텐츠(정책·시설·전법·시나리오 등) 약 7,800여 개 문자열 신규 번역
  - 예: 가와나카지마 전투, 딱따구리 전법, 통치 범위, 성하 방침, 오케하자마 전투, 나가시노 전투 등
- 일반 게임과 공유하는 약 4만 6천여 개 문자열은 기존 번역을 그대로 재사용해 용어 일관성 유지
- 파워업키트 전용 한글 폰트 적용 (일반판 폰트와 별도, 실기 부팅 안전성 확보)

## 포함 내용
- 게임 텍스트 번역 — 메뉴·UI·이벤트 대사·게임 진행·무장 열전·튜토리얼·**파워업키트 신규 콘텐츠**
- 한글 폰트 (서울한강체) — 일반 게임·파워업키트 각각 전용 적용
- 대사 줄바꿈 최적화, 전각→반각 문자 정규화
- 이미지 한글화: 원형 명령 메뉴 · 시스템/내비 버튼 · 전투 군평정 · 튜토리얼 다이어그램 · 합전 배너 · 화면 제목 108종 · 부팅 경고문
- 실행파일(exefs/main) 시스템 메시지 6종 한글화 — 일반 게임·파워업키트 각각 전용 패치

## 적용 대상
- 타이틀: 노부나가의 야망 신생 (일본판) — **파워업키트 적용, 게임 버전 1.1.5**
- 타이틀 ID: 일반 게임 `01007AB012872000` + 파워업키트 `01007AB012872001` (둘 다 필요)

## 폴더 구조 (배포 zip)
```
atmosphere/contents/
  01007ab012872000/          ← 일반 게임 프로그램
    exefs/main                (한글 시스템 메시지, 무압축 NSO — 실기 호환)
    romfs/MSG/JP/               한글 텍스트
    romfs/RES_JP/res_lang.bin   한글 폰트 + 이미지
  01007ab012872001/          ← 파워업키트 프로그램
    exefs/main                 PUK 전용 한글 시스템 메시지
    romfs/MSG_PK/JP/            PUK 한글 텍스트 (정책·시설·전법·시나리오·튜토리얼 등)
    romfs/RES_JP_PK/res_lang_pk.bin  PUK 전용 한글 폰트
```

## 설치 방법 (클린 설치 권장)

**실기 (Atmosphère)**
1. 기존 모드 폴더(`01007ab012872000`, `01007ab012872001`)를 **완전히 삭제**
2. zip 속 `atmosphere` 폴더를 **SD카드 루트에 통째로 복사** (병합)
   → `sd:/atmosphere/contents/01007ab012872000/` 와 `…872001/` 두 폴더가 생깁니다.
3. 게임 실행

**에뮬레이터 (Ryujinx/Yuzu)**
- 게임 우클릭 → Open Mods Directory(또는 Open Mod Data Location) 밑에
  `01007ab012872000/`, `01007ab012872001/` 각 폴더를 만들어 위 구조의 내용을 넣으세요.

## exefs 패치 주의
- `exefs/main`(실행파일)은 **게임 버전 1.1.5(파워업키트) 전용**입니다.
- 부팅에 문제가 생기면 해당 프로그램의 `exefs` 폴더만 삭제하세요. 나머지 한글패치는 그대로 동작합니다.

## 알려진 사항
- 크레딧(스태프 이름)과 시나리오 검색용 내부 인덱스 문자열은 원문(일본어) 유지됩니다.
- 미사용 더미(플레이스홀더) 데이터 일부는 원문으로 남아있을 수 있으나 게임 화면에는 표시되지 않습니다.
- 폰트는 서울한강체(서울시 배포 무료 글꼴)라 원본 붓글씨체와 스타일이 다를 수 있습니다.
- 버튼 아이콘(✔·↩·⊘ 등)은 한글화 과정에서 생략될 수 있으나 라벨 뜻은 그대로입니다.
- 인명·지명은 국립국어원 일본어 표기법을 따릅니다.
- 기계 번역이 포함되어 일부 문장은 어색하거나 오역이 있을 수 있습니다.

## 저장소 구성 (개발/기여용)

이 저장소에는 **패치를 만드는 도구와 번역 데이터**가 함께 들어 있습니다.
게임 원본 데이터·복호화 키는 포함하지 않습니다.

```
tools/            빌드 파이프라인 (Python) — 포맷 디코드/인코드, 폰트 확장,
                  MSG 재조립, 이미지 로컬라이즈, 패키징
  extractor/      병합 romfs 추출기 (LibHac, .NET 6) — 키는 본인 것 사용
  scratch/        1회성 디버그/수정 스크립트 (참고용)
translation/
  source_jp/      일반 게임 원문 문자열 배치 (번역 키) 529개
  korean/         index로 짝지어진 한국어 번역 529개
  source_jp_puk/  파워업키트 신규 원문 배치 45개
  korean_puk/     index로 짝지어진 한국어 번역 45개
docs/
  FORMATS.md      리버스 엔지니어링한 파일 포맷 분석 (LINKDATA/KT/NSO/MSG/G1N/G1T/파워업키트 이중 프로그램)
  BUILD.md        빌드 가이드
```

빌드 방법은 [docs/BUILD.md](docs/BUILD.md), 포맷 분석은 [docs/FORMATS.md](docs/FORMATS.md) 참고.
