# 노부나가의 야망 신생 한글 패치 (비공식)

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.
폰트에 한글 2,400여 자를 추가하고 게임 텍스트(메뉴·UI·이벤트 대사·파워업키트 신규
콘텐츠 포함) 전체를 한국어로 번역합니다.

## 게임 버전에 맞는 zip 하나만 받으세요

최신 릴리스에는 **게임 버전별 zip이 함께 담겨 있습니다.** 본인 게임 버전에 맞는 zip
하나만 받아 설치하면 됩니다.

| 게임 버전 | 받을 파일 | 내용 |
|---|---|---|
| **1.1.7** (파워업키트) | `NobunagaShinsei_KR_for_1.1.7_PUK.zip` | 최신. 「결전」 시스템 번역 + **한국어 입력기** 포함 |
| **1.1.5** (파워업키트) | `NobunagaShinsei_KR_for_1.1.5_PUK.zip` | 1.1.5 전용 |
| **1.1.4** (일반판, 파워업키트 미적용) | `NobunagaShinsei_KR_for_1.1.4_vanilla.zip` | 구버전 전용 |

> ⚠️ **게임 버전을 반드시 맞춰 주세요.** 특히 `exefs/main`(실행파일)은 게임 버전 전용이라
> 다른 버전에 얹으면 부팅에 문제가 생길 수 있습니다. 부팅이 안 되면 해당 폴더의 `exefs`만
> 지우면 나머지 한글패치(텍스트·폰트·이미지)는 그대로 동작합니다.

아래 변경 이력은 파워업키트(1.1.5/1.1.7) 기준입니다.

## 한국어 입력기 (1.1.7) — 무장 이름 한글 입력

**1.1.7 zip 한정**으로, 무장을 새로 만들거나 이름을 편집할 때 **한글을 직접 입력**할 수
있습니다. 커뮤니티 기여([이슈 #1](../../issues/1), 제작 @2seunghee)로 만들어진 실행파일
패치를 반영한 것으로, 성명·독음의 글자 수 검증을 우회해 **기존 무장 편집 시 "이름 6자
이상"으로 저장되지 않던 문제**도 함께 해결됩니다.

- 실행파일(`exefs/main`)을 수정하므로 **1.1.7 파워업키트 전용**입니다(1.1.5·1.1.4 미적용).
- 원본 xdelta를 공식 xdelta3(체크섬 검증)로 우리 1.1.7 main에 적용한 뒤, 의존성 없는
  복사/삽입 패치로 재표현해 빌드에 통합 — **이후 1.1.7 릴리스에도 자동 포함**됩니다.
- 도구: `tools/apply_hangul_kbd.py`, `tools/hangul_kbd_117.patch.json`,
  출처·검증·범위는 `tools/NOTE_hangul_keyboard.md` 참고

## 상황 설명 나레이션 줄바꿈 자동화 (1.1.5 / 1.1.7)

시나리오 중간중간 나오는 **상황 설명 나레이션**이 한글 번역 길이 때문에 화면 오른쪽으로
넘치던 문제를 해결했습니다. 나레이션의 수동 줄바꿈(`\n`)을 제거해 게임이 박스 폭에 맞춰
자동으로 줄을 나누도록 했습니다.

- 나레이션·이벤트 컷신 텍스트(`msgev`)는 **전체폭 자동 줄바꿈 박스**라 줄바꿈 제거가 안전
  (이벤트 스크립트 분석 + JP 원문 줄 폭 분포로 확인)
- **초상화·이름표가 붙는 캐릭터 대사(`msggame`)는 원래 줄바꿈을 유지** — 이 박스는 자동
  줄바꿈이 아니라 한 줄을 폭에 맞춰 글자를 축소하므로, 한 줄로 합치면 글자가 작아집니다
- 운문·명문(사세구·한시·와카) 등 조판이 중요한 7건은 적대적 다중 에이전트 검증으로 가려내
  줄바꿈 유지
- 도구: `tools/strip_msgev_only.py`, `tools/protect_msgev.json`,
  `tools/NOTE_dialogue_linebreak.md`

## v3.8 — 분할 대사 결합 품질 개선

게임 대사는 내부적으로 여러 **조각(fragment)**으로 쪼개진 뒤 실행 시 이어 붙는
구조라, 각 조각이 따로 번역되며 경계에서 어색해지던 문장을 다듬었습니다. 사용자
스크린샷 사례를 기반으로 전수 검사했습니다.

- 말더듬 오역(뒷단어 첫소리 반복): 「처, 천하를」→「천, 천하를」, 「저, 적습」→「적, 적습」

  「공략 세력<u>인</u>」, 「조정<u>에</u>」→「조정<u>에서</u>」 등
- 일본어 원문 1:1 대조로 최소 수정, 조각 개수·서식 토큰 보존. 총 17개 문자열.
- 도구: `tools/extract_split_chains.py`(분할 사슬 추출), `tools/survey_awkward.py`(패턴

## v3.5 — 용어 표기 일관성 개선

다른 한글화 프로젝트([parsifal295/nobu16-korean-patch](https://github.com/parsifal295/nobu16-korean-patch))의 공개 용어집을 참고해 제 번역의 용어 선택을 점검하고,
더 자연스럽거나 정확한 쪽으로 개선했습니다. **대사 문장 자체를 가져온 것이 아니라,
고유명사·게임 용어를 어떻게 한글로 옮길지에 대한 표기 관례만 비교**했습니다 — 실제
개선은 제 판단으로 결정했습니다.

- `姫`(히메): "희" → **"공주"** (표준어가 아닌 낯선 한자음 대신 명확한 단어로)
- `勲功`: "훈공" → **"공훈"** (어순 수정)
- `一向宗`/`乾坤`: 음역(잇코종/겐콘) → **"일향종"/"건곤"** (기존 한자음으로 통일)
- `労力`(시설 건설용 인력 자원): "노력" → **"노동력"** (일상어 "노력(effort)"과 혼동 방지)
- `郡代`/`城代`: "군대"/"성대" → **"군 대관"/"성 대관"** ("군대(軍隊)"/"성대(vocal cords)"와의
  동음이의어 충돌 회피)

원문 JP 키를 기준으로 범위를 한정해 적용했으며(무관한 동음이의 단어는 건드리지 않음),
총 약 330개 문자열이 영향을 받았습니다. 전체 번역의 절대다수는 기존과 동일합니다.

또한 릴리스 전 검증으로 **빌드 산출물 전체(기본판 3개 + 파워업키트 7개 bin,
DLC 19개 파일)를 일본어 원본과 약 10만 문자열 단위로 전수 대조**했습니다. 결과:
화면에 노출되는 미번역 문자열 0건 (크레딧·검색용 요미가나·미사용 더미 슬롯은
정책대로 원문 유지). 이 과정에서 줄바꿈 토큰 불일치로 검증에서 탈락하던 번역
항목 2건(`b0175_dialog`, `r9001_dialog`)도 정비했습니다.

- 원형 명령 메뉴 하단 상태 배지: **中止(중지)**, **失敗(실패)**, **達成(달성)**

이 라벨들은 표준 버튼 모양과 달라(회전된 배지, 다른 크기) 원래의 버튼 자동 검출
목록(`e5_btns.json`)에 포함되지 않아 누락됐던 것입니다.

## v3.1 — 파워업키트 대사창 줄바꿈 오버플로 수정

v3.0에서 파워업키트 텍스트(§ v3.0 항목)를 새로 넣으면서, 예전에 일반 게임에
적용했던 "대사가 3줄 박스를 넘겨 잘리는 문제" 수정이 파워업키트 쪽에는 빠져
있었습니다. 원인은 새로 삽입한 한글 글리프가 자간이 조여지지 않은(원본 전각
글자 자간을 그대로 물려받은) 상태였던 것 — 일반 게임 폰트보다 훨씬 넓게
렌더링되어 대사가 쉽게 넘쳤습니다. 파워업키트 폰트 자간을 일반 게임과 동일한
방식으로 조이고, 대사(`msgev.bin`)의 박스-오버플로 줄바꿈도 함께 제거해
엔진이 자동으로 재배치하도록 복원했습니다(742건 수정).

## v3.0 — 파워업키트 대사·메뉴 텍스트 완전 한글화

이전 버전까지는 이미지(원형 명령 메뉴, 시스템 버튼 등)만 한글이고 **파워업키트의
대사·메뉴 텍스트는 일본어**로 나오는 문제가 있었습니다.

**원인**: 파워업키트는 일반 게임과 **별도의 실행 프로그램**(타이틀 ID
`01007AB012872001`, 일반 게임의 `01007AB012872000`과는 다름)이며, 텍스트를 일반판과
다른 전용 파일(`MSG_PK/`, `RES_JP_PK/`)에서 읽습니다. 게다가 이 PUK 프로그램은
독자적인 폰트 리소스(`res_lang_pk.bin`)를 쓰는데, 폰트 로더가 **원본과 정확히 같은
압축 해제 크기**를 요구해서 — 일반판에 쓰던 방식(폰트를 통째로 더 큰 한글 폰트로
교체)을 그대로 적용하면 **부팅 전에 죽습니다.** v3.0은 이 두 파일 세트를 모두
찾아내 정확한 방식(미사용 전각 글자 자리에 한글을 크기 불변으로 제자리 삽입)으로
한글화했습니다. 자세한 내용은 [docs/FORMATS.md](docs/FORMATS.md#7-파워업키트-872001-이중-프로그램-구조)를
참고하세요.

- 파워업키트 신규 콘텐츠(정책·시설·전법·시나리오 등) 약 7,800여 개 문자열 신규 번역
  - 예: 가와나카지마 전투, 딱따구리 전법, 통치 범위, 성하 방침, 오케하자마 전투, 나가시노 전투 등
- 일반 게임과 공유하는 약 4만 6천여 개 문자열은 기존 번역을 그대로 재사용해 용어 일관성 유지
- 파워업키트 전용 한글 폰트 적용 (일반판 폰트와 별도, 실기 부팅 안전성 확보)

## 폴더 구조 (배포 zip)

```
atmosphere/contents/

    romfs/RES_JP/res_lang.bin   한글 폰트 + 이미지
  01007ab012872001/          ← 파워업키트 프로그램

    romfs/MSG_PK/JP/            PUK 한글 텍스트 (정책·시설·전법·시나리오·튜토리얼 등)
```

## 설치 방법 (클린 설치 권장)

**실기 (Atmosphère)**
1. 기존 한글패치 모드 폴더(`01007ab012872000`, `01007ab012872001`, `01007ab0128730xx`)를
   **완전히 삭제**
2. 받은 zip 속 `atmosphere` 폴더를 **SD카드 루트에 통째로 복사** (병합)
   → `sd:/atmosphere/contents/…` 아래에 해당 폴더들이 생깁니다.
3. 게임 실행

**에뮬레이터 (Ryujinx/Yuzu)**
- 게임 우클릭 → Open Mods Directory(또는 Open Mod Data Location) 밑에 zip 속
  `atmosphere/contents/<타이틀 ID>/` 안의 `romfs`·`exefs`를 넣으세요.

## exefs 패치 주의
- `exefs/main`(실행파일)은 **받은 zip의 게임 버전 전용**입니다(1.1.7 zip은 1.1.7용, 1.1.5 zip은
  1.1.5용, 1.1.4 zip은 1.1.4용). 다른 버전에 얹지 마세요.
- 부팅에 문제가 생기면 해당 프로그램의 `exefs` 폴더만 삭제하세요. 나머지 한글패치(텍스트·
  폰트·이미지)는 그대로 동작합니다. 단, 1.1.7의 **한국어 입력기는 `exefs`를 지우면 함께 빠집니다.**

## 알려진 사항
- 크레딧(스태프 이름)과 시나리오 검색용 내부 인덱스 문자열은 원문(일본어) 유지됩니다.
- 미사용 더미(플레이스홀더) 데이터 일부는 원문으로 남아있을 수 있으나 게임 화면에는 표시되지 않습니다.
- 폰트는 서울한강체(서울시 배포 무료 글꼴)라 원본 붓글씨체와 스타일이 다를 수 있습니다.
- 버튼 아이콘(✔·↩·⊘ 등)은 한글화 과정에서 생략될 수 있으나 라벨 뜻은 그대로입니다.
- 인명·지명은 국립국어원 일본어 표기법을 따릅니다.
- 기계 번역이 포함되어 일부 문장은 어색하거나 오역이 있을 수 있습니다.
- (1.1.7) 한국어 입력기는 이름/독음 글자 수 **검증을 우회**하므로, 게임 기본 제한을 벗어난
  이름도 저장될 수 있습니다.
