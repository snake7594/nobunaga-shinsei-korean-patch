# 노부나가의 야망 신생 한글 패치 (비공식) — v0.9 (부분 번역)

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.
폰트에 한글 2,400여 자를 추가하고 게임 텍스트를 한국어로 번역합니다.

> ⚠️ **부분 번역본입니다.** 대사·이벤트·UI 위주로 약 1.8만 문장이 번역되었으며,
> 무장 열전·일부 데이터는 아직 일본어로 표시됩니다. 미번역 부분은 원문(일본어)이 그대로 나오며
> 게임 진행에는 문제가 없습니다.

## 적용 대상
- 타이틀: 노부나가의 야망 신생 (일본판)
- 타이틀 ID: `01007AB012872000`

## 설치 방법

### Ryujinx
1. 게임 목록에서 해당 게임을 **우클릭 → Open Mods Directory**
2. 열린 폴더 안에 이 패치의 `romfs` 폴더를 통째로 복사
   (최종 경로: `mods\contents\01007ab012872000\romfs\MSG\...`)
3. 게임 실행

### Yuzu 계열
1. 게임 **우클릭 → Open Mod Data Location**
2. 새 폴더(예: `KoreanPatch`)를 만들고 그 안에 `romfs` 폴더 복사

### 실기 (Atmosphère)
- SD카드의 `atmosphere\contents\01007AB012872000\romfs\` 에 `MSG`, `RES_JP` 폴더 복사

## 포함 파일
- `romfs/RES_JP/res_lang.bin` — 한글 글리프가 추가된 게임 폰트 (KS X 1001 2,350자 + 실사용 음절)
- `romfs/MSG/JP/strdata.bin` — 시스템/UI/데이터 텍스트
- `romfs/MSG/JP/ev_strdata.bin` — 이벤트 대사
- `romfs/MSG/JP/msggame.bin` — 게임 진행 메시지

## 알려진 사항
- 미번역 문자열은 일본어로 표시됩니다(정상).
- 폰트는 서울한강체(서울시 배포 무료 글꼴)로 렌더링되어 원본 붓글씨체와 스타일이 다를 수 있습니다.
- 인명·지명은 국립국어원 일본어 표기법을 따릅니다.
