# 노부나가의 야망 신생 한글 패치 (비공식) — v1.5

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.

게임 버전은 일본 일반판 1.1.4 버전입니다. DLC는 아직 적용하지 않았습니다.

폰트에 한글 2,400여 자를 추가하고 게임 텍스트를 한국어로 번역합니다.

> **번역 완성도: 유니크 문자열 기준 100%** (46,871 / 46,871).
> 메뉴·UI·이벤트 대사·게임 진행 메시지·무장 열전·명품 해설·튜토리얼까지 전부 번역되었습니다.
>
> **v1.2**: 이미지 한글화 1차 — 붓글씨 화면 제목 이미지 108종(지행·평정·출진·외교 등)과
> 부팅 경고문 화면을 서울한강체 기반 한글 이미지로 교체.
> **v1.3**: 화면 제목 라벨이 위/아래/옆으로 잘리던 문제를 수정(잉크를 셀 안 여백에 맞춰 배치).
> **v1.4**: 실행파일(exefs/main)에 하드코딩된 일본어 시스템 메시지 6종을 한글화
> (튜토리얼 확인·세이브 손상·DLC 없음·그래픽 초기화 오류 등).
> **v1.5**: 메인 화면 원형 명령 메뉴(평정·내정·임명·군사·외교)가 일본어로 남던 문제 대응 —
> 세 언어(JP/SC/TC) 텍스트 슬롯 모두에 한글을 넣어 어느 슬롯을 읽든 한글로 표시되게 함.

## exefs 패치 주의
- v1.4부터 `exefs/main`(실행파일)이 포함됩니다. romfs와 함께 모드 폴더에 복사하세요.
- 게임 버전이 다르면 실행파일이 맞지 않아 부팅이 안 될 수 있습니다(현재 대상: 일본 일반판 1.1.4).
  부팅에 문제가 있으면 `exefs` 폴더만 삭제하면 나머지 한글패치는 그대로 동작합니다.

## 적용 대상
- 타이틀: 노부나가의 야망 신생 (일본판)
- 타이틀 ID: `01007AB012872000`

## 설치 방법

### Ryujinx
1. 게임 목록에서 해당 게임을 **우클릭 → Open Mods Directory**
2. 열린 폴더 안에 zip 속 `NobunagaShinsei_KR` 폴더를 통째로 복사
   (최종 경로: `mods\contents\01007ab012872000\NobunagaShinsei_KR\romfs\MSG\...`)
3. 게임 실행

### Yuzu 계열
1. 게임 **우클릭 → Open Mod Data Location**
2. 새 폴더(예: `KoreanPatch`)를 만들고 그 안에 `NobunagaShinsei_KR\romfs` 폴더 복사

### 실기 (Atmosphère)
- SD카드의 `atmosphere\contents\01007AB012872000\romfs\` 에 `MSG`, `RES_JP` 폴더 복사

## 포함 파일
- `romfs/RES_JP/res_lang.bin` — 한글 글리프가 추가된 게임 폰트 (KS X 1001 2,350자 + 실사용 음절)
- `romfs/MSG/JP/strdata.bin` — 시스템/UI/데이터/무장 열전 텍스트
- `romfs/MSG/JP/ev_strdata.bin` — 이벤트 대사
- `romfs/MSG/JP/msggame.bin` — 게임 진행 메시지·튜토리얼
