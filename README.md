# 노부나가의 야망 신생 한글 패치 (비공식) — v3.7

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.

> **이 패치는 게임 버전 1.1.5(파워업키트 적용) 전용입니다.**
> 파워업키트가 적용되지 않은 구버전(1.1.4)을 지원하던 옛 릴리스는 저장소 히스토리에는
> 남아 있지만 더 이상 유지하지 않습니다. 앞으로는 1.1.5(파워업키트) 기준으로만 관리합니다.

폰트에 한글 2,400여 자를 추가하고 게임 텍스트(메뉴·UI·이벤트 대사·파워업키트 신규
콘텐츠 포함) 전체를 한국어로 번역합니다.

## v3.5 — 용어 표기 일관성 개선

다른 한글화 프로젝트([parsifal295/nobu16-korean-patch](https://github.com/parsifal295/nobu16-korean-patch))의 공개 용어집을 참고해 제 번역의 용어 선택을 점검하고,
더 자연스럽거나 정확한 쪽으로 개선했습니다. **대사 문장 자체를 가져온 것이 아니라,
고유명사·게임 용어를 어떻게 한글로 옮길지에 대한 표기 관례만 비교**했습니다 — 실제
개선은 제 판단으로 결정했습니다.

- `姫`(히메): "희" → **"공주"** (표준어가 아닌 낯선 한자음 대신 명확한 단어로)
- `勲功`: "훈공" → **"공훈"** (어순 수정)
- `一向宗`/`乾坤`: 음역(잇코종/겐콘) → **"일향종"/"건곤"** (기존 한자음으로 통일)
- `労力`(시설 건설용 인력 자원): "노력" → **"노동력"** (일상어 "노력(effort)"과 혼동 방지)
- `郡代`/`城代`: "군대"/"성대" → **"군 대관"/"성 대관"** ("군대(軍隊)"/"성대(vocal cords)"와의
  동음이의어 충돌 회피)

원문 JP 키를 기준으로 범위를 한정해 적용했으며(무관한 동음이의 단어는 건드리지 않음),
총 약 330개 문자열이 영향을 받았습니다. 전체 번역의 절대다수는 기존과 동일합니다.

또한 릴리스 전 검증으로 **빌드 산출물 전체(기본판 3개 + 파워업키트 7개 bin,
DLC 19개 파일)를 일본어 원본과 약 10만 문자열 단위로 전수 대조**했습니다. 결과:
화면에 노출되는 미번역 문자열 0건 (크레딧·검색용 요미가나·미사용 더미 슬롯은
정책대로 원문 유지). 이 과정에서 줄바꿈 토큰 불일치로 검증에서 탈락하던 번역
항목 2건(`b0175_dialog`, `r9001_dialog`)도 정비했습니다.

## v3.4 — 파워업키트 "역사 인물 크로스오버" DLC 텍스트 한글화

기존에는 못 찾았던 **별도 파일 형식(DLC_PK/JP/\*.n16)**에 담긴 파워업키트 크로스오버
콘텐츠를 찾아 번역했습니다 — 미나모토노 요시쓰네·항우·송강·칭기즈 칸·오키타 소지·
가수 히카와 기요시, 6명의 역사 인물 게스트 무장과 그들의 특성·if 시나리오

이 파일들은 MSG_PK와도 res_lang_pk 이미지와도 다른 **완전히 별도의 텍스트 저장
포맷**(`.n16`)이라 이전까지는 조사 대상에서 빠져 있었습니다. 자세한 포맷 분석은
[docs/FORMATS.md](docs/FORMATS.md)의 "DLC_PK — 역사 인물 크로스오버 콘텐츠" 항목 참고.

## v3.3 — 파워업키트 전용 이미지(정책·시설 라벨, 특성/가보 이름) 한글화

파워업키트 전용 리소스(`res_lang_pk.bin`)에만 들어있던 이미지 76종을 새로 찾아
한글화했습니다:
- **정책/시설/편집 관련 라벨 43종** — 恩賞(은상)·捕縛登用(포박 등용)·家宰特性選択(가재
  특성 선택)·評定衆(평정중)·シナリオ編集(시나리오 편집) 등 PUK 메뉴·툴팁 라벨
- **특성/가보 이름 배지 11종**(연출 상태별 변형 포함 33개 텍스처) — 武田之赤備(다케다의
  적비)·百折不撓(백절불굴)·日本号(니혼고)·無双槍(무쌍창) 등

이 라벨들은 투명 배경 위에 텍스트만 있는 텍스처라, 기존 인페인팅 방식(주변 색을
번져서 채움)을 그대로 쓰면 지운 자리에 얼룩진 잔상이 남는 문제가 있어 — 알파 채널을
완전히 투명하게 지우는 방식으로 새로 처리했습니다.

**알려진 한계**: PUK 튜토리얼/도움말에 쓰이는 정적 스크린샷(날짜·자원량 등 실제
게임 상태가 찍힌 캡처 이미지)은 이번에 포함되지 않았습니다 — 실제 플레이 화면에는
해당 용어들이 이미 번역된 텍스트로 표시되며, 캡처 이미지 자체를 다시 그리는 건
범위 밖입니다.

## v3.1 — 파워업키트 대사창 줄바꿈 오버플로 수정

v3.0에서 파워업키트 텍스트(§ v3.0 항목)를 새로 넣으면서, 예전에 일반 게임에
적용했던 "대사가 3줄 박스를 넘겨 잘리는 문제" 수정이 파워업키트 쪽에는 빠져

렌더링되어 대사가 쉽게 넘쳤습니다. 파워업키트 폰트 자간을 일반 게임과 동일한
방식으로 조이고, 대사(`msgev.bin`)의 박스-오버플로 줄바꿈도 함께 제거해
엔진이 자동으로 재배치하도록 복원했습니다(742건 수정).

## v3.0 — 파워업키트 대사·메뉴 텍스트 완전 한글화

이전 버전까지는 이미지(원형 명령 메뉴, 시스템 버튼 등)만 한글이고 **파워업키트의
대사·메뉴 텍스트는 일본어**로 나오는 문제가 있었습니다.

**원인**: 파워업키트는 일반 게임과 **별도의 실행 프로그램**(타이틀 ID
`01007AB012872001`, 일반 게임의 `01007AB012872000`과는 다름)이며, 텍스트를 일반판과
다른 전용 파일(`MSG_PK/`, `RES_JP_PK/`)에서 읽습니다. 게다가 이 PUK 프로그램은
독자적인 폰트 리소스(`res_lang_pk.bin`)를 쓰는데, 폰트 로더가 **원본과 정확히 같은
압축 해제 크기**를 요구해서 — 일반판에 쓰던 방식(폰트를 통째로 더 큰 한글 폰트로
교체)을 그대로 적용하면 **부팅 전에 죽습니다.** v3.0은 이 두 파일 세트를 모두
찾아내 정확한 방식(미사용 전각 글자 자리에 한글을 크기 불변으로 제자리 삽입)으로
한글화했습니다. 자세한 내용은 [docs/FORMATS.md](docs/FORMATS.md#7-파워업키트-872001-이중-프로그램-구조)를
참고하세요.

- 파워업키트 신규 콘텐츠(정책·시설·전법·시나리오 등) 약 7,800여 개 문자열 신규 번역
  - 예: 가와나카지마 전투, 딱따구리 전법, 통치 범위, 성하 방침, 오케하자마 전투, 나가시노 전투 등
- 일반 게임과 공유하는 약 4만 6천여 개 문자열은 기존 번역을 그대로 재사용해 용어 일관성 유지
- 파워업키트 전용 한글 폰트 적용 (일반판 폰트와 별도, 실기 부팅 안전성 확보)

## 설치 방법 (클린 설치 권장)

**실기 (Atmosphère)**
1. 기존 모드 폴더(`01007ab012872000`, `01007ab012872001`)를 **완전히 삭제**
2. zip 속 `atmosphere` 폴더를 **SD카드 루트에 통째로 복사** (병합)
   → `sd:/atmosphere/contents/01007ab012872000/` 와 `…872001/` 두 폴더가 생깁니다.
3. 게임 실행

**에뮬레이터 (Ryujinx/Yuzu)**
- 게임 우클릭 → Open Mods Directory(또는 Open Mod Data Location) 밑에
  `01007ab012872000/`, `01007ab012872001/` 각 폴더를 만들어 위 구조의 내용을 넣으세요.

## exefs 패치 주의
- `exefs/main`(실행파일)은 **게임 버전 1.1.5(파워업키트) 전용**입니다.
- 부팅에 문제가 생기면 해당 프로그램의 `exefs` 폴더만 삭제하세요. 나머지 한글패치는 그대로 동작합니다.

## 알려진 사항
- 크레딧(스태프 이름)과 시나리오 검색용 내부 인덱스 문자열은 원문(일본어) 유지됩니다.
- 미사용 더미(플레이스홀더) 데이터 일부는 원문으로 남아있을 수 있으나 게임 화면에는 표시되지 않습니다.
- 폰트는 서울한강체(서울시 배포 무료 글꼴)라 원본 붓글씨체와 스타일이 다를 수 있습니다.
- 버튼 아이콘(✔·↩·⊘ 등)은 한글화 과정에서 생략될 수 있으나 라벨 뜻은 그대로입니다.
- 인명·지명은 국립국어원 일본어 표기법을 따릅니다.
- 기계 번역이 포함되어 일부 문장은 어색하거나 오역이 있을 수 있습니다.

## 저장소 구성 (개발/기여용)

이 저장소에는 **패치를 만드는 도구와 번역 데이터**가 함께 들어 있습니다.
게임 원본 데이터·복호화 키는 포함하지 않습니다.

```
tools/            빌드 파이프라인 (Python) — 포맷 디코드/인코드, 폰트 확장,
                  MSG 재조립, 이미지 로컬라이즈, 패키징
  extractor/      병합 romfs 추출기 (LibHac, .NET 6) — 키는 본인 것 사용
  scratch/        1회성 디버그/수정 스크립트 (참고용)
translation/
  source_jp/      일반 게임 원문 문자열 배치 (번역 키) 529개
  korean/         index로 짝지어진 한국어 번역 529개
  source_jp_puk/  파워업키트 신규 원문 배치 45개
  korean_puk/     index로 짝지어진 한국어 번역 45개
docs/
  FORMATS.md      리버스 엔지니어링한 파일 포맷 분석 (LINKDATA/KT/NSO/MSG/G1N/G1T/파워업키트 이중 프로그램)
  BUILD.md        빌드 가이드
```

빌드 방법은 [docs/BUILD.md](docs/BUILD.md), 포맷 분석은 [docs/FORMATS.md](docs/FORMATS.md) 참고.

### 기여
번역 개선은 `translation/korean/` 또는 `translation/korean_puk/` 의 `t` 값을 고쳐 PR 해주세요.
(서식 토큰 `%s %d \n [bNNN]`, `<ESC>C?` 컬러 태그는 보존 필수)
