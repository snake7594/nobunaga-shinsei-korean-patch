# 노부나가의 야망 신생 한글 패치 (비공식) — v2.1

Switch용 **노부나가의 야망 신생(信長の野望・新生)** 일본판의 비공식 한글 패치입니다.

게임 버전은 **일본 일반판 1.1.4**입니다. DLC는 아직 적용하지 않았습니다.

폰트에 한글 2,400여 자를 추가하고 게임 텍스트를 한국어로 번역합니다.

> **v2.1 — 메인 화면 원형 명령 메뉴(評定·内政·任命·軍事·外交…)를 드디어 한글로!**
> 그동안 원형 명령 휠이 계속 일본어로 남았던 **진짜 원인**을 찾았습니다. 이 라벨들은
> 텍스트(strdata)가 아니라 **`RES_JP/res_lang.bin` 안에 그림(텍스처)으로 구워져 있던
> 명령 버튼 아틀라스**였습니다. (이 리소스는 아카이브 구조상 그동안 도구가 읽지 못하던
> 숨은 항목이라 발견이 늦었습니다.) v2.1은 이 아틀라스의 명령 버튼 **약 50종(색상 상태
> 포함 380개 버튼)**을 서울한강체 한글로 교체했습니다. 픽토그램·구름 버튼·회전 화살표는
> 원본 그대로 보존합니다. 이제 원형 명령 메뉴와 각종 명령 버튼이 한글로 표시됩니다.

## 포함 내용
- 게임 텍스트 100% 번역 (메뉴·UI·이벤트 대사·게임 진행·무장 열전·튜토리얼)
- 한글 폰트 2,400여 자 (서울한강체)
- **원형/명령 버튼 아틀라스 한글화 (신규, v2.1)** — 평정·내정·임명·군사·외교·군개발·
  영지제책·지행·대관·전봉·출진·성하시설·부대정보 등 약 50종
- 붓글씨 화면 제목 이미지 108종 + 부팅 경고문 한글화
- 실행파일 시스템 메시지 6종 한글화 (1.1.4 기준)

## 적용 대상
- 타이틀: 노부나가의 야망 신생 (일본판), **버전 1.1.4**
- 타이틀 ID: `01007AB012872000`

## 설치 방법 (클린 설치 권장)
1. 기존 모드 폴더를 **완전히 삭제**
2. 게임 목록에서 해당 게임을 **우클릭 → Open Mods Directory**
3. zip 속 `NobunagaShinsei_KR` 폴더를 통째로 복사
4. 게임 실행

Yuzu 계열은 우클릭 → Open Mod Data Location, 실기(Atmosphère)는
`atmosphere/contents/01007AB012872000/`에 `romfs`·`exefs` 복사.

## exefs 패치 주의
- `exefs/main`(실행파일)은 **게임 버전 1.1.4 전용**입니다.
- 부팅에 문제가 생기면 `exefs` 폴더만 삭제하세요. 나머지 한글패치는 그대로 동작합니다.

## 아직 일본어로 남는 부분 (추후 작업 예정)
같은 방식으로 그림에 구워진 일본어라 아직 남아 있는 화면이 있습니다:
- 전투 군평정 화면(軍評定·勝利·敗北·戦功)
- 튜토리얼 흐름 다이어그램(知行·内政·戦 등)
- 합전 시작 배너(長篠の戦い 등 전투명)
- 타이틀 로고(信長の野望 新生 — 상표라 보존 검토)
